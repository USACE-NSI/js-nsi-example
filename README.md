# js-nsi-example

A simple javascript exmple for downloading structures from the NSI API

To run open a terminal in vscode and type:
```
node index.js
```

The expected results is:
```
Downloading structures from: https://nsi.sec.usace.army.mil/nsiapi/structures?bbox=-81.58418,30.25165,-81.58161,30.26939,-81.55898,30.26939,-81.55281,30.24998,-81.58418,30.25165
Structure count 2735
Total Value: $234,084.65
```
